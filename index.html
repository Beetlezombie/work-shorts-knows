Starting a new project

Folder structure: Dist\CSS | Dist\img | Dist\js | SCSS\main.scss | SCSS\_config.scss etc

Create a .gitignore file using terminal command: touch .gitignore. Add for example node_modules

Git & Github

Adding everything to a local repo

Initializing a new project: git init

Adding everything to the git (in the staging area): git add .

Committing the staging area to the repo: git commit -m 'Comment'

Pushing to Github (remote)

Node $ Node-SASS

For being able to use node-saas, you need a json file. Via terminal command npm init

Answer the questions, and a package.json file is created. Anything installed by npm is put in this folder

To install node-SASS use command npm i node-sass

To set up watch script do the following: "sass": node-sass -w scss/ -o dist/css/ --recursive"

To run watch script use: npm run sass
